Use shears on vines to stop them from growing! This allows vines to be sheared exactly like cave (glowberry) vines!

## * Note about server compatibility *

Because this mod modifies block state, while playing on a server the mod needs to be installed both on the server AND the client! If you hop on to a server and notice all the blocks are wrong, either you have the mod and the server doesn't or vice-versa.

I've been investigating ways around this, but I don't have anything promising yet. For now, you'll simply have to install the mod in both places. Sorry!
